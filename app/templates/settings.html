📋 How to Login to TikTok

Method 1 — Browser Cookie Export (for VPS):

  1. Install EditThisCookie extension di Chrome
  2. Buka tiktok.com → Login ke akun kamu
  3. Klik icon EditThisCookie → Export (icon kotak)
  4. Paste isinya di textarea dibawah → klik Upload Cookies

Method 2 — Playwright Login (for local dev):

cd app && python -c "from uploader import save_session; save_session()"

Ini akan membuka browser, login manual, lalu tekan Enter untuk save cookies.
